# DynamicGrip

For further details: https://www.nexusmods.com/skyrimspecialedition/mods/110031?tab=description

Built from https://github.com/libxse/commonlibsse-template

### Requirements
* [XMake](https://xmake.io) [2.8.2+]
* C++23 Compiler (MSVC, Clang-CL)

## Getting Started
```bat
git clone https://github.com/bosnThs/dynamicGrip
cd dynamicGrip
.\BuildRelease.bat
```
